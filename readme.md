# Rock, paper, scissors

Game created in OOP with Javascript, BEM and SCSS. Webpack bundled.

# How to run

1. Clone the repository.

2. `npm i`

3. `npm start` for start Webpack dev server.

4. `npm run build` for build production. Path: `dist/`.
